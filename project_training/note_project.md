# Create React Template

## Steps

1. create a react app called "template"
```
yarn create react-app template
```

## React Files & Folders
- public repo
  - static
    - ex) HTML, images, etc
    - 
- src folder
  - dynamical
  - ex) APP.js

2. Clean the template
- Public Folder
  - index.html
  - delete all except index.html & favicon.ico
  
- src folder
   - APP.js => APP.jsx


3. Starting React APP
```
yarn start
```
